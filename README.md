# Education-in-Danger-Dashboard-2020-2026-
##Project Overview

This project presents an interactive Excel dashboard analyzing incidents affecting education between 2020 and 2026. The analysis highlights patterns of attacks on schools and universities, as well as the impact on students and educators across different countries.

The objective of the project is to transform raw humanitarian data into clear insights using Microsoft Excel data analysis and visualization techniques.

#Dataset

Dataset Source:
Humanitarian Data Exchange (HDX)
https://data.humdata.org/dataset/sind-education-in-danger-monthly-news-briefs-dataset

The dataset contains reported incidents affecting education including:

- Attacks on schools

- Attacks on universities

- Students killed, injured, or kidnapped

- Educators killed, injured, or kidnapped

- Damaged schools

- Countries where incidents occurred

- Year of occurrence

##Key Questions Answered

This dashboard was designed to answer important analytical questions:

- How have education-related attacks changed from 2020–2026?

- Which countries reported the highest number of incidents?

- What trends exist in attacks on schools vs universities?

- How are students and educators affected by these incidents?

- Which countries recorded the highest number of affected children?

#Dashboard Features

The Excel dashboard includes the following visualizations:

• Total Events by Year
• Number of Events by Country
• Attacks on Schools vs Universities Trend
• Students Affected (Killed, Injured, Kidnapped)
• Educators Affected (Killed, Injured, Kidnapped)
• Incident Distribution
• Children Impact by Country
• Damaged Schools Overview

Interactive filters and slicers allow users to explore trends across different countries and years.

#Key Insights

Some insights derived from the analysis include:

 - A significant increase in reported education-related incidents between 2020 and 2025.

- Certain countries consistently reported higher levels of education-related attacks.

- Students appear to be the most affected group, particularly through injury and kidnapping incidents.

- Attacks on schools occur more frequently than attacks on universities.
